# توصيل طلبات

تطبيق توصيل طلبات للسائقين - Flutter Application

## المميزات

- عرض الطلبات المتاحة للتوصيل
- تفاصيل كاملة لكل طلب
- خريطة التوصيل
- تتبع حالة الطلبات
- ملف شخصي للسائق

## البدء

1. تأكد من تثبيت Flutter
2. قم بتشغيل `flutter pub get`
3. شغل التطبيق باستخدام `flutter run`

## المتطلبات

- Flutter SDK >= 3.7.2
- Dart SDK >= 3.0.0
